About this site

This is the WebDB project for CS257 at Carleton College by Yuping Huang. Phineas Callahan contributed to the early effort in building this site

This website allows a search for presidential and governor election data based on state and time range. It groups data from the same county together, thus allowing for analysis of voting behavior changes over time for a given county.

The data come from CQPress Voting and Elections Collection.The licensing agreement is shown on the download data page. Our data are downloaded from the download page.

Features

1. A navigation bar to move between this about page and the main query page.
2. A web form to execute the search based on state and time range.
3. The output data table sorted by county; if one use the same start and end date then it will return the election data in a given year for a given state.
4. Color-coded table to show democrat, republican and third-party votes.
5. Text instructions.
6. I wished I had implemented a secondary search interface where the dropdown list for counties are loaded according
to the state chosen by the user. This will allow finer-grained searches. Sorting the output by county names is my way
of dealing with the lack of this feature.
